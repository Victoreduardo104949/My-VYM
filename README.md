# Execute e implante seu app do AI Studio

Este repositório contém tudo o que você precisa para rodar seu app localmente.

Veja seu app no AI Studio: https://ai.studio/apps/dae1a1a3-05a4-4fe6-a91a-dd330b39ad70

## Rodar Localmente

**Pré-requisitos:** Node.js


1. Instale as dependências:
   `npm install`
2. Defina a `GEMINI_API_KEY` no arquivo [.env](.env) com sua chave da API Gemini
3. Inicie o app:
   `npm run dev`
